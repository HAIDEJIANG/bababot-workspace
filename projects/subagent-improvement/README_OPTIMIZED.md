# LinkedIn 联系人深度分析系统 v3.1 - 全面优化版

## 📋 系统概述

**目标：** 对全部 3,185 位 LinkedIn 联系人进行深度分析，获取公司业务范围、个人职责、日常动态，用于业务拓展

**核心功能：**
- ✅ Profile 数据提取（姓名/公司/职位/地点等）
- ✅ 90 天发帖采集（业务意图识别）
- ✅ 业务线索自动筛选
- ✅ 优先级打分
- ✅ 推荐跟进策略

## ⚠️ 注意事项

### 1. Cookie 配置（关键）
- 需要使用**已登录 Cookie**才能看到完整 Profile
- 未登录状态会显示"Join LinkedIn"
- 建议使用主账号 + 备用账号轮换

### 2. 风控防范
- 间隔时间已优化（30-60 秒）
- 请勿过度频繁访问
- 如遇风控暂停，等待恢复后继续

### 3. 系统资源
- 浏览器内存占用：~500MB
- 进程 CPU 占用：~5-15%
- 输出文件大小：~50-100MB
