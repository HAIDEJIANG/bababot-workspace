### 遇到的问题与解决方案

#### 问题 1：Windows fcntl 模块不支持
- **现象**: `ModuleNotFoundError: No module named 'fcntl'`
- **解决**: 修改 `clawteam/team/tasks.py`，使用 `filelock` 替代

#### 问题 2：WSL 网络代理冲突
- **现象**: apt update 卡住，pip 下载失败
- **解决**: 在 WSL 内部禁用代理，使用阿里云 DNS 和镜像源

#### 问题 3：clawteam 命令不在 PATH
- **现象**: `command not found`
- **解决**: 添加 `~/.local/bin` 到 PATH

#### 问题 4：OpenClaw Node.js 版本要求
- **现象**: 需要 Node.js v22.12+，WSL 默认版本不够
- **解决**: 使用 Windows 版 OpenClaw（通过 /mnt/c/ 路径调用）

### 使用场景建议

#### 适合 ClawTeam 的场景
1. **复杂任务分解** - 如 LinkedIn 全量联系人分析（3,185 位）
2. **多角色协作研究** - 如航空市场分析报告（需要多个专业 agent）
3. **长时间运行任务** - 无超时限制，可运行数小时
4. **需要工作区隔离** - 多个 agent 并行开发不同模块

#### 适合原生 sub-agents 的场景
1. **简单并行任务** - 如批量数据查询
2. **Windows 原生环境** - 无需 WSL 配置
3. **快速原型测试** - 配置简单，启动快
